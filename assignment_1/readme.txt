README

Platform and Compiler
======================
Platform: Arch Linux
Kernel: 4.14.14-1-ARCH

Instructions
=====================
1. To build and run the program, run make and then ./a1.out
2. Each scene can be traversed with the left and right arrow keys.
3. Each level, iteration, or revolution can be increased and decreased with the up and down arrow keys.
4. To close the program, press the ESC key.
