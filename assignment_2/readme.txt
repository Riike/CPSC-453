README
=======

Platform and Compiler
======================
Platform: Arch Linux
Kernel: 4.14.14-1-ARCH

Instructions
=============
1. Type make
2. ./a2.out

Change Image
------------
Left Arrow Key: Previous image
Right Arrow key: Next image

Transformations
---------------
A Key: Rotate Counterclockwise
D Key: Rotate Clockwise
Scroll Wheel: Zoom in/out
Click-and-Drag: Pan up/down/left/right

Effects
--------------
1 Key: Luminance 1
2 Key: Luminance 2
3 Key: Luminance 3
4 Key: Sepia
5 Key: Vertical Sobel
6 Key: Horizontal Sobel
7 Key: Unsharp Mask
8 Key: 3x3 Gaussian
9 key: 5x5 Gaussian
0 key: 7x7 Gaussian
ENTER Key: Original Image
