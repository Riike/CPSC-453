README
=======

Platform and Compiler
======================
Platform: Arch Linux
Kernel: 4.14.14-1-ARCH

Instructions
=============
1. Type make
2. ./a4.out

Change Scene
------------
Left Arrow Key: Previous scene
Right Arrow key: Next scene

Zoom In/ Zoom Out
-------------------------------------
Up Arrow Key: Increase focal length
Down Arrow Key: Decrease focal length
